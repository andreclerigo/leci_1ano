Este conjunto de pastas e ficheiros formam um exemplo de desenvolvimento de um programa que permite fazer cálculos com fracções (números racionais).

Este programa está feito de forma diferente em cada uma das várias versões, cada qual na sua pasta/pacote (package). A versão v1 contém um programa monolítico. As versões seguintes introduzem novos conceitos e mecanismos que permitem melhorar progressivamente a forma como o programa está implementado. O utilizador do programa {@code FractionCalculator} possivelmente não notará qualquer diferença entre as várias versões, mas o programador deverá aperceber-se que de versão para versão ganha em reutilizabilidade, compreensibilidade, extensibilidade, segurança e outras qualidades "internas" que lhe valorizam e facilitam o trabalho.

Como ler este documento:

Este documento é composto de várias páginas HTML e deve ser lido usando um browser qualquer, abrindo o ficheiro {@code doc/index.html}.

  • Avance por ordem para cada versão (package) indicada no topo desta página.
  • No pacote de cada versão encontra uma lista de classes.
  • Comece pela classe do programa {@code FractionCalculator}, depois avance seguindo as ligações indicadas.
  • Em cada classe encontra explicações gerais e detalhes sobre cada atributo e função.
  • Poderá mesmo ver o código associado a cada função seguindo a ligação associada ao nome respectivo.
Em qualquer altura pode regressar à classe, ao pacote ou aqui (Overview) para navegar para qualquer outra página da forma que entender.

Este documento foi gerado com {@code javadoc} (ver abaixo) a partir dos comentários incluídos nos ficheiros {@code .java}. Por isso também pode simplesmente navegar por esses ficheiros usando um editor de texto comum, mas terá mais dificuldade a ler alguns comentários e a seguir as ligações.

Como gerar este documento:

Este documento, mantido na pasta {@code doc/}, foi gerado correndo o script contido no ficheiro {@code makeDoc}. Em Unix, poderá executar directamente este ficheiro com o comando: 


  sh makeDoc

Como compilar os programas:

Supondo que está no directório base do exemplo, pode compilar todos os ficheiros com o comando: 


  javac v*/*.java

Como executar os programas:

Mantendo-se no directório base, execute por exemplo: 


  $ java -ea v1.FractionCalculator 1/5 + 2/6
  (1/5) + (2/6) = (16/30)
